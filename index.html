estrutura basica

!doctype html ( para mostrar ao navegador que é um documento html)

html ( para colocar o conteúdo do site)

head ( que é a parte de configuração do site)

body (que é a parte de conteúdo do site)

sempre entre os conteudos temos que colocar <> para iniciar os codicos e <> com uma / entre os dois < para finalozalos

tags mais utilizadas


o P é uma tag de parágrafo ou texto do site

tag hr serve para criar uma linha horizontal

tag b serve para deixar o texto em negrito

tag br serve para criar uma quebra de linha como se fosse um paragrafo

tag i serve para deixar o texto em itálico

tag u serve para deixar o texto sublinhado

tag mark serve para destacar o texto com uma cor de fundo amarela, como se fosse um marca texto

tag small serve para deixar o texto em tamanho menor

tag del serve para deixar o texto rasurado

tag sup serve para deixar o texto em sobrescrito

tag sub serve para deixar o texto em subscrito

tag h1 é a de título principal do site, mas podem ser usadas outras t ags de título (h2, h3, h4, h5, h6)
que são usadas para organizar o conteúdo do site, sendo h1 o título mais importante e h6 o menos importante,
pense nelas como se fossem os títulos de um livro, onde o h1 seria o título do livro, o h2 seria o título dos capítulos,
o h3 seria o título das seções e assim por diante.

dica para facilitar fazer a estrutura do site

se você estiver programando em VISUAL STUDIO CODE use a tag html:5 para fazer a estrutura basica automaticamente

caso precise alterar o idioma do site, logo a frente do html inicial coloque o atributo lang="pt-br" para português do Brasil

elementos de citação

tag abbr serve para criar uma abreviação e mostrar o texto completo quando o usuário passa o mouse sobre ela
palavra

tag address serve para exibir informações de contato,

como endereço físico ou email local de residência, cidade,
meu.email@gmail.com

tag cite serve para citar uma fonte ou referência como filmes e seriados, ex: One Piece tem mais de 1000 episódios
mesmo eu colocando esta tag, não ira aparecer para as pessoas, mas o navegador reconhece como obra

tag q serve para criar uma citação curta, ex: esta é uma citação curta

tag

serve para criar uma citação longa, ex:

inportante (links)

para colocar um link em um site, usamos a tag a com o atributo href para indicar o endereço do link, ex: Acessar o YouTube
para caso queira colocar que seu link abra em outra aba, use o atributo target="_blank"
